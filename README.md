# HateSpeechDetection
Built various classification models in order to detect which tweet was hate speech or otherwise. 

## About dataset
The data set used for this research was publicly sourced and collected from Kaggle. The data was first published four years ago by Raul Agarwal and has become very popular for hate speech analysis and detection with a total of 34691 views and 5306 downloads. The data set contains two files i.e. the test and train data. The train data set will be used to train the model while the
other will be used to test the accuracy of the model.
